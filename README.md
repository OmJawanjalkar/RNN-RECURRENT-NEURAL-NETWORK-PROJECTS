# RNN Recurrent Neural Network Projects

This repository covers fundamental RNN concepts and includes an **IMDB Movie Review Sentiment Classification** project with a **Streamlit web application**.

## 🚀 Project: End-to-END IMDB Sentiment Classification

A sentiment classification model built using a **Simple RNN** to classify IMDB movie reviews as **Positive** or **Negative**.

### Model Workflow

`Text Review → Sequence Encoding → Padding → Embedding Layer → SimpleRNN → Dense Layer → Sentiment Prediction`

## 🛠️ Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Streamlit

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd RNN-RECURRENT-NEURAL-NETWORK-PROJECTS
```

Create a virtual environment:

```bash
python -m venv .venv
```

Activate the virtual environment:

```bash
.venv\Scripts\activate
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ Run the Application

Run the Streamlit application:

```bash
streamlit run app.py
```
